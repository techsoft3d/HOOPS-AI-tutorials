## Fetch required binaries (Manual, Windows)

1. Go to: https://github.com/TECHSOFT3D/HOOPS-AI-tutorials/releases/tag/v1.0.preview
2. Download the asset(s): `cadfiles.zip, flows.zip, rained_models.zip`.
3. Create `.\packages\` in the repo root and extract the zip there.
   - Right-click → Extract All… → choose the repo’s `packages` folder.
4. The content of the packages folder if fully ignore in git.
